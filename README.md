# Telegram-бот для конвертации .docx в PDF

Бот принимает документы в формате `docx` через Telegram и конвертирует их в `pdf` с помощью LibreOffice. Все операции логируются, а данные о пользователях и конвертациях сохраняются в PostgreSQL.

## Возможности:
- Приём `docx`-файлов от пользователей.
- Конвертация в `pdf` через `soffice --headless`.
- Автоматическая очистка временных файлов.
- Сохранение информации о каждой конвертации:  
  — ID пользователя и файла  
  — статус (`pending`, `processing`, `completed`, `failed`)  
  — временная метка
- Поддержка многопользовательского режима.

## Запуск в Docker:
- Создайте .env файл и внесите в него определение переменных окружения:
  - BOT_TOKEN - токен бота в Telegram
  - POSTGRES_PASSWORD - пароль от БД
  - POSTGRES_USER - имя пользователя БД
  - POSTGRES_DB - название БД
- запустите: ``docker-compose up --build``

## Технологии:
- Python 3.14
  - uv
- Библиотеки:
  - Telegram Bot API (pyTelegramBotAPI)
  - SQLAlchemy + Alembic (ORM и миграции)
  - python-dotenv
- СУБД PostgreSQL
- Docker

## Примечания:
- Сообщения пользователю отображаются на русском языке.
- При ошибках пользователь получает понятное уведомление, а ошибка фиксируется в логах.
